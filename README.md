# Ejercicios de C++

## Ejercicios

### Distancia de Hamming

En este ejercicio se calcula cuántas posiciones son diferentes entre dos cadenas.
Primero se revisa que ambas tengan la misma longitud y después se comparan carácter por carácter para contar las diferencias.

### Números Armstrong

Este programa revisa si un número es **Armstrong**.
Básicamente se separan los dígitos del número, cada uno se eleva a la potencia del número total de dígitos y luego se suman.
Si el resultado es igual al número original, entonces es un número Armstrong.

### Sistema de Parqueadero

Aquí se crea una clase que simula un **parqueadero** con espacios para carros grandes, medianos y pequeños.
Cuando llega un carro, el programa revisa si todavía hay espacio disponible para ese tipo de vehículo.

### Space Age

En este ejercicio se calcula la **edad en diferentes planetas** a partir de una cantidad de segundos.
Primero se convierte a años en la Tierra y después se usa el período orbital de cada planeta para saber cuántos años equivaldrían en Mercurio, Venus, Marte, etc.
